数据库是用来存放 MM-Wiki 系统的一些基础数据

SQLite 是非常小的，轻量级的数据库，SQLite 不需要配置，这意味着不需要安装或管理。如果您的使用场景数据量少，建议选择使用此数据库类型。

SQLite 配置
* SQLite文件位置
添加超级管理员（超级管理员是 MM-Wiki 的最高级别的用户，安装完成后第一次登录使用该用户名和密码登录）
* 只能由数字和字母组成
* 请记住密码用于登录

数据库是用来存放 MM-Wiki 系统的一些基础数据

MySQL 是最流行的，开源的，可处理拥有上千万条记录的大型关系型数据库管理系统，MySQL 需要下载并安装才能使用。如果您熟悉 MySQL，存储数据量较大，建议使用该类型的数据库。

MySQL 配置
* 需自行安装 MySQL
* 须具备表操作权限
* 不存在会自动创建
添加超级管理员（超级管理员是 MM-Wiki 的最高级别的用户，安装完成后第一次登录使用该用户名和密码登录）
* 只能由数字和字母组成
* 请记住密码用于登录
